# 🚀 Fullstack CRUD Application (.NET + React)

## Overview
This project is a full-stack CRUD (Create, Read, Update, Delete) application built to understand and implement modern web development using .NET Framework, Entity Framework, SQL Server, and React.

The application demonstrates how a frontend interacts with a backend API and database to perform complete data management operations.

---

## Project Objective
The goal of this project is to:
- Learn full-stack development
- Understand REST API design using .NET
- Work with databases using Entity Framework (ORM)
- Integrate a React frontend with a .NET backend
- Perform CRUD operations end-to-end

---

## Architecture

Frontend (React)  →  Backend (.NET API)  →  Database (SQL Server)

- React handles UI and user interaction
- .NET handles business logic and APIs
- Entity Framework manages database communication

---

## Tech Stack

### Backend
- .NET Framework / ASP.NET Web API
- Entity Framework (ORM)

### Frontend
- React.js
- Axios (for API calls)

### Database
- SQL Server

### Tools
- Visual Studio
- VS Code

---

## Features

- Create new records
- View all records
- Update existing records

- RESTful API integration
